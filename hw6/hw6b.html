Booklet plugin

Welcome to My Booklet

Hover over the edge of this page you can click to "flip" the page or click-hold on the edge of the new page to "turn" it yourself.

What is the Booklet plugin?

I think the plugin is self-explanatory. It's a little plugin that you can use on your website to create a booklet a user can flip through in their webpage.

How to install Booklet plugin

This is a link to Booklet's installation manual.

The installation is easy to follow and I had it up and running in a few minutes. I also used this style guide when I turned on the option to have a closed book.

How to customize the Booklet plugin

Here's a link to Booklet's documentation page. This has a bunch of options on how to customize your booklet. To name a few options you can: the size of the booklet, speed of the page flip, and what the starting page could be.

You can click on the option methods to see instructions on how to use it as a selector or what method you need to call to able to turn on your options.

What I options used:

  • height: 500
  • width: 700
  • speed: 500
  • closed: true
  • closedFrontTitle: Start
  • pagePadding: 20 -- so you can see the edge better
