What is the TwentyTwenty plugin?

If you've taken those eye exams that optometrists the TwentyTwenty plugin reminds you of when the doctor flips a switch to blur/unblur what you're looking at. If that doesn't quite make sense I believe the TwentyTwenty website says it's like a tool you can use to play spot the difference between before-and-after images.

The TwentyTwenty homepage contains a download for the plugin and the instructions on how to install the plugin.
