My Favourite Games

Guild Wars 2

One of my favourite games to play when I can squeeze enough time away from work and school. There are plenty different modes of game style you can get into like player vs. environment, player vs. player, or world vs. world (which is pvp on a server-wide scale). Content-wise the main story arcs are still being updated on a semi-regular basis so it isn't like you're purchasing a game with just limited content, there's always something new to look forward to.

I really enjoy it.

Link to the main landing page: Guild Wars 2

Reasons to play:

  • Core game is free
  • Expansion pack goes on sale often
  • Friendly gaming community

New CSS3 Techniques

CSS3 Buttons - Cursor selector

I thought this was pretty cool, there's a selector that will control the behaviour of your mouse when you hover over an element on a webpage. Probably only works for anything that has a cursor.

CSS property: cursor

  • pointer: mouse will be a pointer
  • not-allowed: mouse will be a circle with a dash through it like a stop sign

Link to research: CSS3 Buttons on W3C

CSS3 Image Filters

I thought this was pretty cool too and we can now apply filters using CSS properties instead of editing in a photo editing program before saving a copy. I added 3 options below but there are a lot more where you can choose from.

The filter property is not supported in Internet Explorer, Edge 12, or Safari 5.1 and earlier. And you have to use the -webkit- prefix and no prefix at the bottom.

CSS property: filter

  • blur: blurs the image
  • brightness: adds brightness to your image
  • greyscale: greyscale your image
